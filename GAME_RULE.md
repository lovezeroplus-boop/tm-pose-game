# GAME_RULE.md

## 1. 개요
*   **게임 이름**: Catch Zone
*   **게임 장르**: 포즈 인식 기반 아케이드 게임
*   **목표**: 화면 상단에서 떨어지는 과일을 바구니로 받아 점수를 획득하고, 폭탄을 피하며 최대한 오래 버티는 것.

---

## 2. 구역 정의 (Zones)
화면은 가로로 3개의 구역으로 나뉩니다.
*   **LEFT (좌)**: 화면 왼쪽 영역
*   **CENTER (중)**: 화면 중앙 영역
*   **RIGHT (우)**: 화면 오른쪽 영역

---

## 3. 조작 방식 (Controls)
플레이어의 웹캠 포즈를 인식하여 바구니(Player)를 이동시킵니다.
*   **왼쪽 기울기/이동 포즈** → 바구니를 **LEFT** 구역으로 이동
*   **정면/중립 포즈** → 바구니를 **CENTER** 구역으로 이동
*   **오른쪽 기울기/이동 포즈** → 바구니를 **RIGHT** 구역으로 이동

---

## 4. 아이템 종류 (Items)
모든 아이템은 화면 상단에서 생성되어 아래로 떨어집니다.

| 아이콘 | 이름 | 유형 | 효과 |
|:---:|:---:|:---:|:---|
| 💣 | **폭탄 (Bomb)** | 장애물 | 닿으면 즉시 **Game Over** |
| 🍎 | **사과 (Apple)** | 점수 | **+100점** |
| 🍐 | **배 (Pear)** | 점수 | **+150점** |
| 🍊 | **오렌지 (Orange)** | 점수 | **+200점** |

---

## 5. 게임 진행 규칙

### 기본 플레이
*   아이템은 3개 구역(LEFT, CENTER, RIGHT) 중 랜덤한 한 곳에서 생성되어 떨어집니다.
*   아이템이 바닥에 닿기 전에 바구니가 해당 구역에 위치해 있으면 획득 판정(Catch)됩니다.
*   폭탄을 획득하면 즉시 게임이 종료됩니다.

### 놓침(Miss) 규칙
과일(점수 아이템)을 받지 못하고 바닥으로 떨어뜨린 경우:
*   **1회 놓침**: 경고 (Life 1 감소)
*   **2회 놓침**: **Game Over** (총 Life: 2개)

---

## 6. 레벨 시스템 (Level System)
게임은 20초 단위로 레벨이 올라가며 난이도가 상승합니다.

*   **진행 방식**: 게임 오버 없이 20초를 버티면 자동으로 다음 레벨로 진입
*   **초기 설정 (Level 1)**:
    *   **Drop Time (낙하 속도)**: 2.0초 (화면 상단에서 하단까지 걸리는 시간)
*   **레벨 업 효과**:
    *   **속도 증가**: 레벨마다 Drop Time이 **0.2초씩 감소**
    *   **생성 주기**: Drop Time의 **60% ~ 80%** 간격으로 아이템 랜덤 생성

---

## 7. UI 요소 (User Interface)
화면에 표시해야 할 정보:
1.  **점수 (Score)**: 현재 획득한 총 점수
2.  **레벨 (Level)**: 현재 진행 중인 단계
3.  **남은 시간 (Time)**: 다음 레벨까지 남은 시간 (20초 카운트다운)
4.  **생명 (Life)**: 남은 기회 표시 (아이콘 또는 하트 형태)
5.  **게임 오버 화면**: 최종 점수 및 재시작 버튼
